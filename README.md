# 🎮 Pygame Platformer

2D platformer built with Python + Pygame.

## Features
- Camera system
- Enemy AI (ghosts, birds, walkers)
- Bullet system with reload mechanic
- Animated HUD (ammo, timer)
- Save system (JSON)
- 2 worlds + boss (in progress)

## How to run
pip install pygame
python main.py

## Built with
- Python 3
- Pygame
